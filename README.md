# MediFlow Hub — Demo

This is a demo/hackathon-ready Next.js + Tailwind project showcasing an inter-department hospital workflow platform.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

The app uses an in-memory mock backend and a simple pub/sub for realtime updates. Use `/patient` for the patient portal, `/dashboard` for staff.
